# Telegram Casino Bot 🎰

Многофункциональный Telegram-бот для азартных игр с интеграцией CryptoPay, реферальной системой и админ-панелью.

## 🌟 Особенности
- 🎮 Несколько игровых режимов (Кости, Слоты, Пакеты)
- 💰 Система пополнения/вывода через CryptoPay
- 👥 Реферальная система с бонусами
- 🎁 Адвент-календарь и ежедневные бонусы
- 👑 Система префиксов и сундуков
- 📊 Полноценная админ-панель
- 📈 Статистика и логирование действий
- 🔒 Гибкая система блокировок

## ⚙️ Технологии
- Python 3.11+
- aiogram 3.x
- SQLite3
- aiocryptopay
- Pillow (для работы с изображениями)

## 🚀 Установка

### Требования:
- Python 3.11 или новее
- Telegram API токен
- CryptoPay API токен

1. Клонировать репозиторий:
git clone https://github.com/devpaket/PaketBet.git
cd telegram-casino-bot

2. Установить зависимости:
pip install -r requirements.txt

3. Запустить инсталлятор:
python install.py

4. Запустить бота:
python main.py



🛠 Команды админа
/admin - Панель управления

/info [ID] - Информация о пользователе

/send [текст] - Рассылка сообщений

/delete [ID] - Удаление пользователя

/restart - Перезагрузка бота

📌 Важно
Для работы с изображениями поместите файлы в папку res/

Регулярно делайте бэкап баз данных

Настройте права доступа для разных уровней администрации
